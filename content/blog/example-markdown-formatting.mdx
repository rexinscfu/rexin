# Markdown Formatting Example

This post demonstrates all the markdown formatting options available in our blog posts. Use this as a reference when writing your own content.

## Headings

You can use up to six levels of headings:

# Heading 1 (reserved for post title)
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

## Text Formatting

You can make text **bold**, *italic*, or ***both***. You can also use ~~strikethrough~~ for crossed-out text.

## Lists

### Unordered Lists

- Item 1
- Item 2
  - Nested item 1
  - Nested item 2
- Item 3

### Ordered Lists

1. First item
2. Second item
   1. Nested item 1
   2. Nested item 2
3. Third item

## Code

### Inline Code

Use `inline code` for small code snippets.

### Code Blocks

```javascript
// JavaScript example
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Hello, ${name}!`;
}

// Call the function
greet('World');
```

```c
// C example
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

```rust
// Rust example
fn main() {
    println!("Hello, World!");
    
    // Variables and types
    let x: i32 = 42;
    let y = 3.14;
    
    println!("x = {}, y = {}", x, y);
}
```
